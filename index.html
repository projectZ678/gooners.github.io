To edit the leaderboard, you need a Personal Access Token with repo write access.

How to get a token:
1. Go to github.com/settings/tokens
2. Click "Fine-grained tokens" → Generate new token
3. Name: gooning-leaderboard
4. Repository: Only your repo
5. Permissions: Contents → Read and write
6. Copy the token → paste here
